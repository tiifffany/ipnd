Resumo de aula

Rede mundial de computadores

A internet é a maior rede de computadores do mundo. Basicamente, a internet utiliza o HTTP (Hypertext Transfer Protocol) para se comunicar com o usuário e seu navegador, e os servidores que compõem a web. Um navegador é um programa que é executado no computador do usuário que exibe arquivos encontrados na web e os servidores são computadores que hospedem todos os arquivos que compõem a web.


Elementos básicos de HTML

HTML significa Hypertext Markup Language e a maioria do conteúdo na internet são arquivos HTML composto de elementos HTML. Em outras palavras, esses elementos são o que usamos para dizer ao navegador como o conteúdo de uma página deve aparecer. Grande parte dos elementos HTML contém uma tag de abertura e uma tag de fechamento. O elemento HTML que não necessita uma tag de fechamento é chamado de void. Dois elementos básicos de HTML são:

< b > conteúdo < /b > que deixa o texto em negrito;
< em > conteúdo < /em > que deixa o texto em itálico.


Atributos de HTML

Nas tags de abertura, é possível adicionar atributos, que são valores que dão informações a mais para o navegador de web. Alguns atributos básicos são:

< a href="site" > link < /a > a tag anchor puxa o atributo href com o valor site e o chama de link;
< img src="site" alt="texto" > a tag image mostra a imagem hospedada no valor site do atributo src. Se não for possível mostrar a imagem, o valor texto do atributo alt é exibido.


Outros elementos de HTML

Antes de apresentar outros elementos de HTML, vale ressaltar que existem dois tipos de elementos: inline (alinhado) e block (cria uma espécie de bloco invísível e editável em volta do conteúdo).

As tags a, img, br, span e strong são exemplos de elementos alinhados.
As tags p, div e form são exemplos de elementos em bloco.
